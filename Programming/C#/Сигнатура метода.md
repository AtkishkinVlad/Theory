#  Сигнатура метода

## Что это такое ?

```markdown
1. Кол-во параметров
2. Типы параметров
3. Порядок параметров
4. Модификаторы параметров
5. Имя метода
```

## А что с названием параметров ?

Название параметров `не входят` в сигнатуру метода

## А можно пример ?

Можно, конечно)

```c#
public int Sum(int x, int y) => x + y;
```

У данного метода сигнатурой будет `Sum (int, int)`
